Featured Projects

These are my own demo projects to show off some of my skills:


Orbit: 3D ISS Tracker

Tech Stack: React, Three.js (React Three Fiber), Tailwind CSS, Live REST API

A high-performance 3D visualization web application that tracks the International Space Station in real-time. Features a fully rendered 3D Earth using high-resolution NASA textures, dynamic lighting, and live positional data (latitude, longitude, velocity) fetched automatically every 2 seconds.

GrooveBox

Tech Stack: Vue 3 (Composition API), Pinia, Tone.js, Tailwind CSS

A browser-based 16-step drum machine and sequencer. Demonstrates complex state synchronization by combining Vue's reactivity system (via Pinia) with the Web Audio API to synthesize sounds dynamically in real-time, completely sidestepping the need for static audio files.

Live Demo →

NeonType

Tech Stack: Vanilla JavaScript, HTML5, CSS3

A cyberpunk-themed speed typing game built without any frameworks to demonstrate raw DOM manipulation skills. The game dynamically generates and styles individual elements on the fly, calculating Words Per Minute (WPM) and typing accuracy using a custom game loop.

Live Demo →

PixelForge

Tech Stack: Vue 3, Tailwind CSS

A retro 16x16 pixel art studio tool that bridges complex array data with advanced CSS mechanics. Users can paint on an interactive grid, and the application instantly compiles the artwork into a single, massive CSS box-shadow string, allowing the art to be rendered via a single HTML <div>.

Live Demo →

Liquid Particles Typography

Tech Stack: Vanilla JavaScript, HTML5 Canvas, CSS3

An interactive typography visualization built entirely from scratch using the HTML5 Canvas API. The application renders hidden text, reads the raw pixel data using getImageData, and translates those pixels into thousands of individual particles. It features custom physics logic to calculate distance, velocity, and friction, allowing the particles to react dynamically to the user's mouse movements.

Live Demo →

Obsidian: Kinetic UI & Physics Engine

Tech Stack: Vanilla JavaScript, HTML5, CSS3

A premium, dependency-free landing page designed to showcase advanced DOM manipulation and mathematical UI interactions. Features a custom physics-driven cursor using linear interpolation (Lerp), magnetic buttons that calculate vector distances to break grid layouts dynamically, and a highly performant scroll-reveal engine built from scratch using the IntersectionObserver API.

Live Demo →

Schematic: Node-Based Visual Editor

Tech Stack: Vanilla JavaScript (ES6), HTML5 SVG, CSS3

An interactive, node-based visual programming interface built entirely without external libraries. The application features an infinite panning workspace, independent drag-and-drop nodes with precise event propagation handling, and a custom rendering engine that calculates and draws dynamic SVG cubic Bezier curves in real-time to simulate physical data routing and state connections.
